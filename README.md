# nginx_pulsion
Nginx management though pulsion scripts

## Install
    pulsion install

## Test the setup
    pulsion doctor

## Start
    pulsion start

## Stop
    pulsion stop

## Restart
    pulsion restart

## Update the nginx config
    pulsion update_config

## Show last access logs
    pulsion logs_access

## Show last error logs
    pulsion logs_error

## Show last access logs on virtual host default
    pulsion logs_default_access
